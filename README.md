# .vim

## Description
vim customizations by Yugal Jindle

## Installation

```
cd ~
mkdir .vim
cd .vim
git clone git@github.com:yugal/dotvim.git .

# Add symlink
ln -s ~/.vim/vimrc ~/.vimrc
```
